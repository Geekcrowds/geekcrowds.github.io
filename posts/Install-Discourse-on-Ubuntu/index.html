Install Discourse on Ubuntu

Install Discourse Dependencies

As regular user run this script 103 in terminal, to setup Rails development environment:-

1

bash <(wget -qO- https://raw.githubusercontent.com/discourse/install-rails/master/linux)

This will install following new packages on your system:

In case you have any of this package pre-installed and donâ€™t want to run entire script, see the script 103 and pick the packages you donâ€™t have currently installed. The script is fine-tuned for Discourse, and includes all the packages required for Discourse installation.

Now that we have installed Discourse dependencies, letâ€™s move on to install Discourse itself. Clone Discourse

Clone the Discourse repository in ~/discourse folder:

1

git clone https://github.com/discourse/discourse.git ~/discourse

~ indicates home folder, so Discourse source code will be available in your home folder.

Setup Database

Create role with the same name as your ubuntu system username:

1

sudo -u postgres createuser -s "$USER"

Bootstrap Discourse

Switch to your Discourse folder:

1

cd ~/discourse

Install the needed gems

1
2

source ~/.bashrc
bundle install

Now that you have successfully installed gems, run these commands:

1
2
3

bundle exec rake db:create 
bundle exec rake db:migrate
RAILS_ENV=test bundle exec rake db:create db:migrate

Try running the specs:

1

bundle exec rake autospec

All the tests should pass.

Start rails server:

1

bundle exec rails server

You should now be able to connect with your Discourse app on http://localhost:3000 959 - try it out!

Starting with Discourse 2.5+ EmberCLI is required in development and these additional steps will be required:

In a separate terminal instance, navigate to your discourse folder (cd ~/discourse) and run:

1

bin/ember-cli

You should now be able to navigate to http://localhost:4200 63 to see your local Discourse installation. Create New Admin

To create a new admin, run the following command:

1

RAILS_ENV=development bundle exec rake admin:create

Follow the prompts, and a new admin account will be created. Configure Mail

Run MailHog:

1

mailhog

Congratulations! You are now the admin of your own Discourse installation!
